# QR-Code-Generator

This is a simple QR Code Generator web application built using HTML, CSS, and JavaScript.

## Features
- Generate QR code from text or URL
- Simple and clean UI
- Fast QR generation
- Works in browser only

## Tech Used
- HTML
- CSS
- JavaScript
- QR Code JS Library

## How to run
1. Download repo
2. Open index.html in browser
3. Enter text or URL
4. QR code will generate
